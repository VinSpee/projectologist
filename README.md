## Dependencies:
- nodejs 
- yeoman
- compass
- singularity
- modular scale

## Install:

0. `git clone https://github.com/VinSpee/projectologist.git; cd projectologist`
1. `npm install yo grunt-cli bower`
2. `sudo gem install compass`
3. `sudo gem install modular-scale`
4. `sudo gem install singularitygs`
5. `npm install`
6. `bower install`
7. `grunt server`

# YOU WIN!
